# Interactive MCQ Quiz App

A clean, modern, and highly responsive Multiple Choice Questions (MCQ) Quiz Application built using vanilla web technologies. The quiz features dynamic question shuffling and a strict per-question countdown timer.

## 🚀 Features

- **Dynamic Shuffling:** Questions and multiple-choice choices are randomized using JavaScript on every load/restart.
- **Countdown Timer:** A built-in 10-second rule for each question to keep the quiz challenging.
- **Instant Visual Validation:** Correct choices light up in green; wrong selections display in red while highlighting the actual right answer.
- **Final Analytics Scoreboard:** Offers a calculated score breakdown at the wrap-up screen with a seamless "Restart" layout.
- **Modern UI & Responsive Design:** Uses smooth gradient tones alongside fully fluid components adaptive across desktops, tablets, and smartphones.

## 📂 Project Architecture

├── index.html       # Structural layouts, wrappers, and core markup
├── style.css        # Aesthetic modifications, typography configurations, and UI states
└── script.js        # Data arrays, core timers, state controllers, and DOM engines

## ⚙️ Tech Stack Employed

- **Markup:** HTML5 Standard
- **Styling:** CSS3 (Flexbox, CSS Gradients, Custom Reset Rules)
- **Fonts:** Google Fonts Integration (Poppins Architecture)
- **Engine Logic:** Vanilla JavaScript (ES6 Execution)

## 🛠️ Instructions to Deploy Locally

1. Clone or download the source files inside a unified operating folder.
2. Ensure file tracking names exactly match: `index.html`, `style.css`, and `script.js`.
3. Launch `index.html` directly into any modern web browser (Chrome, Edge, Firefox, or Safari).
